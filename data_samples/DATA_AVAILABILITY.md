# Data Availability

This repository contains only small synthetic or redacted samples.

Included:

- aggregate sample metrics for the runnable evidence-loop demo;
- public documentation of metric definitions and failure categories.

Withheld:

- raw simulator traces;
- checkpoints;
- internal run logs;
- private platform paths;
- large files, caches, videos, and tensorboard artifacts;
- internal QA, audit, sanitization, revision, or review-bundle materials.

The withheld files are excluded because they are large, noisy, private, or not suitable for a public project showcase. Public materials are limited to claim-safe summaries and runnable diagnostic examples.
